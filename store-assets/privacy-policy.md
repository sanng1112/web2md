# Privacy Policy for Web2md

**Last updated:** June 2026

## Data Collection

Web2md does **not** collect, transmit, store, or share any personal information or browsing data.

All content conversion happens **entirely client-side** within your browser. The extension:

- Does not send any data to external servers
- Does not use analytics or tracking services
- Does not embed any third-party telemetry
- Does not read, modify, or transmit data from any website you visit beyond what is necessary for the HTML-to-Markdown conversion

## Required Permissions

The extension requests the following permissions solely for core functionality:

| Permission | Purpose |
|---|---|
| `activeTab` | Access the current tab's HTML content for conversion |
| `contextMenus` | Provide right-click "Convert to Markdown" functionality |
| `storage` | Save user preferences (theme, extraction mode, export options) locally |
| `clipboardWrite` | Support "Auto-copy to clipboard" feature |
| `<all_urls>` | Allow conversion on any website |

## Local Storage

User preferences (theme, extraction mode, output options) are saved using Chrome's `storage.local` API. This data never leaves your browser.
