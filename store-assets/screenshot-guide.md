# Screenshot Guide for Chrome Web Store

Screenshots must be exactly **1280x800** PNG. Take these 4 screenshots:

## 1. Popup UI (`screenshot-1.png`)
- Open the extension popup on an article page (e.g. medium.com, dev.to)
- Show Smart Extract mode selected, options visible, preview filled with Markdown
- Stats bar showing char/word count

## 2. Right-click menu (`screenshot-2.png`)
- Right-click on a web page background
- Show context menu with "Convert page to Markdown" highlighted

## 3. Dark mode (`screenshot-3.png`)
- Toggle dark mode on the popup
- Show dark theme with Markdown preview

## 4. Output comparison (`screenshot-4.png`)
- Show the Markdown output preview with YAML frontmatter, headings, links, and code blocks
- Make sure the preview has enough content to look impressive

## How to take screenshots on Chrome

1. Open `chrome://extensions/` and enable Developer mode
2. Load the unpacked extension
3. Open the extension popup, resize the inspector if needed
4. Use OS screenshot tool: `Ctrl+Shift+S` (Windows) or `Cmd+Shift+4` (Mac) for region capture
5. Crop to exactly 1280x800 using any image editor
