# Web2md ‚Äî Store Listing

## Tagline
Convert any web page to clean Markdown in one click. Preserves links, images, tables, code blocks, headings ‚Äî with smart content extraction and dark mode.

## Description (long)

Web2md is the fastest way to turn any web page into clean, well-structured Markdown. Whether you're a developer saving documentation, a writer archiving research, or anyone who needs to capture web content ‚Äî Web2md delivers.

### Why Web2md?

Most "Save as Markdown" tools give you a noisy dump of the entire page ‚Äî navigation, ads, sidebars, and all. Web2md uses **Mozilla Readability** (the same engine behind Firefox Reader View) to intelligently extract just the article content, then converts it to beautiful Markdown with **Turndown.js**.

### What you get

- **Clean, readable Markdown** with proper headings, links, images, and formatting
- **YAML frontmatter** with title, source URL, description, publication date, and featured image
- **Two extraction modes** ‚Äî Smart (Readability) for articles, Raw for full-page fidelity
- **Right-click anywhere** ‚Äî convert the full page or just your text selection
- **Keyboard shortcut** ‚Äî `Ctrl+Shift+M` / `Cmd+Shift+M` for instant conversion
- **Dark mode** ‚Äî toggle between light and dark themes
- **Multiple exports** ‚Äî download as `.md`, clean `.html`, or plain `.txt`
- **Image control** ‚Äî include or exclude images in your output
- **Word count** ‚Äî see char/word/read-time stats for every conversion
- **Auto-copy** ‚Äî optionally copy to clipboard on every conversion

### Perfect for

- **Developers** ‚Äî save API docs, READMEs, and technical articles for offline reference
- **Writers & researchers** ‚Äî archive web articles with source attribution preserved
- **Students** ‚Äî capture lecture notes, research papers, and references
- **Content creators** ‚Äî repurpose web content into blog drafts or notes
- **Obsidian / Notion users** ‚Äî pipe clean Markdown directly into your knowledge base

### Privacy-first

Web2md runs **entirely in your browser**. No data is ever sent to any server. No analytics. No tracking. Your web content stays on your machine.

## Permissions explained

| Permission | Why it's needed |
|---|---|
| `activeTab` | Read the current page to convert its content |
| `contextMenus` | Add right-click "Convert to Markdown" menu |
| `storage` | Save your preferences locally |
| `clipboardWrite` | Auto-copy Markdown to clipboard |
| `<all_urls>` | Enable conversion on any website |
